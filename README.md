# AI Document Assistant 🤖📄

## Overview

AI Document Assistant is a Generative AI-powered web application that enables users to upload documents and interact with them intelligently. The application leverages Google's Gemini AI model to analyze document content, answer questions, generate summaries, and provide insights from uploaded files.

Built using Streamlit, the project offers a user-friendly interface for document processing and AI-assisted information retrieval.

---

## Features

* 📄 Upload and process documents
* 🤖 AI-powered document understanding using Google Gemini
* 📝 Generate document summaries
* ❓ Ask questions about uploaded documents
* 🔍 Extract key information and insights
* 🌐 Interactive Streamlit web interface
* ⚡ Fast and efficient document analysis

## Installation

### Clone the Repository

```bash
git clone https://github.com/ArunRam12-tech/AI_Document_Assistant.git
cd AI_Document_Assistant
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file in the project root:

```env
GOOGLE_API_KEY=your_api_key_here
```

### Run the Application

```bash
streamlit run app.py
```
